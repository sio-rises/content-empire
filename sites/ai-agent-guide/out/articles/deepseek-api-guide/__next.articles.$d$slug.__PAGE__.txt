2:T1a73,<h2>DeepSeek APIとは？</h2>
<p>DeepSeekは中国のAI企業が開発した大規模言語モデルで、APIを通じてテキスト生成やチャット機能を利用できます。OpenAIのGPTシリーズに似た性能を持ちながら、<strong>圧倒的に低い料金</strong>が特徴です。この記事ではDeepSeek APIの基本的な使い方、料金体系、設定方法を詳しく解説します。</p>
<h2>なぜDeepSeek APIが注目されているのか？</h2>
<li><strong>低コスト</strong>：OpenAIの約1/10以下の料金で同等の性能を実現</li>
<li><strong>高速応答</strong>：軽量モデルによる高速な推論</li>
<li><strong>日本語対応</strong>：日本語の理解・生成に優れる</li>
<li><strong>簡単な移行</strong>：OpenAI APIと互換性のあるエンドポイント</li>
<h2>DeepSeek APIの始め方</h2>
<h3>1. アカウント作成とAPIキーの取得</h3>
<li><a href="https://platform.deepseek.com/" target="_blank" rel="noopener">DeepSeek Platform</a>にアクセス</li>
<li>メールアドレスまたはGoogleアカウントで登録</li>
<li>ログイン後、ダッシュボードから「API Keys」メニューを選択</li>
<li>「Create API Key」ボタンをクリックしてキーを生成</li>
<li>キーを安全な場所に保存（<strong>一度しか表示されません</strong>）</li>
<h3>2. 環境設定</h3>
<p>Pythonを使用する場合、<code>openai</code>ライブラリをインストールします（DeepSeekはOpenAI互換のため）。</p>
<pre><code class="language-bash">pip install openai
</code></pre>
<p>環境変数にAPIキーを設定：</p>
<pre><code class="language-bash">export DEEPSEEK_API_KEY="your-api-key-here"
</code></pre>
<h3>3. 基本的な使い方</h3>
<p>#### チャット補完（Chat Completion）</p>
<pre><code class="language-python">from openai import OpenAI
<p>client = OpenAI(
    api_key="your-api-key-here",
    base_url="https://api.deepseek.com/v1"
)</p>
<p>response = client.chat.completions.create(
    model="deepseek-chat",  # または deepseek-coder
    messages=[
        {"role": "system", "content": "あなたは親切なアシスタントです。"},
        {"role": "user", "content": "DeepSeek APIの使い方を教えてください。"}
    ],
    temperature=0.7,
    max_tokens=1024
)</p>
<p>print(response.choices[0].message.content)
</code></pre></p>
<p>#### テキスト補完（Legacy）</p>
<pre><code class="language-python">response = client.completions.create(
    model="deepseek-chat",
    prompt="DeepSeek APIの特徴は？",
    max_tokens=200
)
</code></pre>

<h2>料金体系</h2>
<p>DeepSeek APIの料金は非常に競争力があります。2026年4月時点の価格：</p>
<table><thead><tr><th>モデル</th><th>入力（1Mトークンあたり）</th><th>出力（1Mトークンあたり）</th></tr></thead><tbody><tr><td>deepseek-chat</td><td>$0.14</td><td>$0.28</td></tr>
<tr><td>deepseek-coder</td><td>$0.14</td><td>$0.28</td></tr>
<tr><td>deepseek-reasoner</td><td>$0.55</td><td>$2.19</td></tr></tbody></table>
<strong>OpenAIとの比較</strong>（GPT-4o miniの場合：入力$0.15、出力$0.60）
<li>チャットモデル：入力はほぼ同額、出力は<strong>約半分</strong></li>
<li>推論モデル：GPT-4oと比較すると<strong>約1/10</strong>のコスト</li>
<p><strong>無料枠</strong>：新規登録時に$10分のクレジットが付与されます（期間限定の可能性あり）。</p>
<h2>詳細設定とパラメータ</h2>
<h3>主要パラメータ</h3>
<table><thead><tr><th>パラメータ</th><th>型</th><th>デフォルト</th><th>説明</th></tr></thead><tbody><tr><td>temperature</td><td>float</td><td>1.0</td><td>0〜2。低いほど決定論的</td></tr>

<p><strong>Q: レート制限は？</strong>
A: 無料枠では1分間に60リクエスト程度。有料プランではさらに緩和されます。</p>
<p><strong>Q: データは学習に使われる？</strong>
A: API経由のデータは学習に使用しないと明記されています。プライバシーに配慮。</p>
