2:T174b,<h2>OpenCodeとは？</h2>
<p>OpenCodeは、AIを活用したコード生成・編集ツールです。特にWebアプリ開発において、Next.jsやReactと組み合わせることで、効率的に開発を進められます。本記事では、OpenCodeを使ったWebアプリ開発の始め方を、具体的な手順とともに解説します。</p>
<h2>1. 環境準備</h2>
<h3>必要なツール</h3>
<li>Node.js (v18以上推奨)</li>
<li>npm または yarn</li>
<li>Git</li>
<li>テキストエディタ（VSCode推奨）</li>
<h3>OpenCodeのインストール</h3>
<p>OpenCodeはCLIツールとして提供されています。以下のコマンドでインストールします。</p>
<pre><code class="language-bash">npm install -g @opencode/cli
</code></pre>
<p>インストール後、バージョンを確認します。</p>
<pre><code class="language-bash">opencode --version
</code></pre>
<h2>2. Next.jsプロジェクトの作成</h2>
<p>OpenCodeはNext.jsとの親和性が高いため、まずはNext.jsのプロジェクトを作成します。</p>
<pre><code class="language-bash">npx create-next-app@latest my-app --typescript
cd my-app
</code></pre>
<h2>3. OpenCodeの初期化</h2>
<p>プロジェクトディレクトリでOpenCodeを初期化します。</p>
<pre><code class="language-bash">opencode init
</code></pre>
<p>これにより、<code>.opencode</code>ディレクトリが作成され、設定ファイルが生成されます。</p>
<h2>4. AIコーディングの基本</h2>
<p>OpenCodeの最大の特徴は、AIによるコード生成です。以下のように、自然言語で指示を出すだけでコードを生成できます。</p>
<h3>例：Todoリストコンポーネントの作成</h3>
<pre><code class="language-bash">opencode generate "ReactのTodoリストコンポーネントを作成。追加、削除、完了の機能を持つ。TypeScriptで書く。"
</code></pre>
<p>OpenCodeは、<code>components/TodoList.tsx</code> にコードを生成します。</p>
<h2>5. 生成されたコードの確認と修正</h2>
<p>生成されたコードを確認し、必要に応じて修正します。OpenCodeはコードの品質が高いですが、人間のレビューは必須です。</p>
<pre><code class="language-tsx">// components/TodoList.tsx
import React, { useState } from 'react';
<p>interface Todo {
  id: number;
  text: string;
  completed: boolean;
}</p>
<p>const TodoList: React.FC = () =&gt; {
  const [todos, setTodos] = useState&lt;Todo[]&gt;([]);
  const [input, setInput] = useState('');</p>
<p>const addTodo = () =&gt; {
    if (input.trim() === '') return;
    setTodos([...todos, { id: Date.now(), text: input, completed: false }]);
    setInput('');
  };</p>
<p>const toggleTodo = (id: number) =&gt; {
    setTodos(todos.map(todo =&gt;
      todo.id === id ? { ...todo, completed: !todo.completed } : todo
    ));
  };</p>
<p>const deleteTodo = (id: number) =&gt; {
    setTodos(todos.filter(todo =&gt; todo.id !== id));
  };</p>
<p>return (
    &lt;div&gt;
      &lt;h1&gt;Todo List&lt;/h1&gt;
      &lt;input
        type="text"
        value={input}
        onChange={(e) =&gt; setInput(e.target.value)}
        onKeyDown={(e) =&gt; e.key === 'Enter' &amp;&amp; addTodo()}
      /&gt;
      &lt;button onClick={addTodo}&gt;追加&lt;/button&gt;
      &lt;ul&gt;
        {todos.map(todo =&gt; (
          &lt;li key={todo.id}&gt;
            &lt;span
              style={{ textDecoration: todo.completed ? 'line-through' : 'none' }}
              onClick={() =&gt; toggleTodo(todo.id)}
            &gt;
              {todo.text}
            &lt;/span&gt;
            &lt;button onClick={() =&gt; deleteTodo(todo.id)}&gt;削除&lt;/button&gt;
          &lt;/li&gt;
        ))}
      &lt;/ul&gt;
    &lt;/div&gt;
  );
};</p>
<p>export default TodoList;
</code></pre></p>
<h2>6. アプリへの組み込み</h2>
<p>生成したコンポーネントを、<code>pages/index.tsx</code> に組み込みます。</p>
<pre><code class="language-tsx">// pages/index.tsx
import type { NextPage } from 'next';
import TodoList from '../components/TodoList';
<p>const Home: NextPage = () =&gt; {
  return (
    &lt;div&gt;
      &lt;h1&gt;My App&lt;/h1&gt;
      &lt;TodoList /&gt;
    &lt;/div&gt;
  );
};</p>
<p>export default Home;
</code></pre></p>
<h2>7. 開発サーバーの起動</h2>
<pre><code class="language-bash">npm run dev
</code></pre>
<p>ブラウザで <code>http://localhost:3000</code> にアクセスし、Todoリストが動作することを確認します。</p>
<h2>8. OpenCodeの高度な使い方</h2>
<h3>コンテキストを指定したコード生成</h3>
<p>プロジェクトの既存コードをコンテキストとして与えることで、より適切なコードを生成できます。</p>
<pre><code class="language-bash">opencode generate "このプロジェクトのスタイルに合わせて、ヘッダーコンポーネントを作成" --context ./styles/globals.css
</code></pre>
<h3>リファクタリング</h3>
<p>既存のコードを改善するよう指示することも可能です。</p>
<pre><code class="language-bash">opencode refactor ./components/TodoList.tsx "パフォーマンスを最適化し、useCallbackとuseMemoを使用"
</code></pre>
<h2>9. デプロイ</h2>
<p>Next.jsアプリはVercelに簡単にデプロイできます。</p>
<li>GitHubリポジトリにコードをプッシュ</li>
<li>Vercelアカウントを作成し、リポジトリをインポート</li>
<li>自動デプロイ</li>
<p>または、以下のコマンドでビルドします。</p>
<pre><code class="language-bash">npm run build
</code></pre>
