f:T1a55,<h1>OpenCodeで始めるWebアプリ開発入門｜Next.jsとReactでAIコーディング</h1>
<p>AIコーディングツールが急速に進化する中、<strong>OpenCode</strong>はオープンソースで高機能な開発環境として注目を集めています。本記事では、OpenCodeを使ってWebアプリを開発する方法を、Next.jsとReactを例に詳しく解説します。初心者でもステップバイステップで進められるよう、環境構築からAI支援機能の活用までをカバーします。</p>
<h2>OpenCodeとは？</h2>
<p>OpenCodeは、AIアシスタントを内蔵したコードエディタです。VS Codeをベースに、コード生成・補完・リファクタリングなどのAI機能をシームレスに利用できます。特筆すべきは、ローカルLLMにも対応しており、プライバシーを保ちながらAIの恩恵を受けられる点です。</p>
<h3>主な特徴</h3>
<li>コード補完・生成（GPT-4やClaude、ローカルLLM対応）</li>
<li>チャットインターフェースによる対話的な開発</li>
<li>ターミナル統合、Git連携</li>
<li>拡張機能豊富（VS Code互換）</li>
<h2>環境構築</h2>
<h3>1. OpenCodeのインストール</h3>
<p>公式サイト（https://opencode.ai）からOSに合わせたインストーラをダウンロードします。macOSの場合はHomebrewでもインストール可能です。</p>
<pre><code class="language-bash">brew install --cask opencode
</code></pre>
<h3>2. Node.jsの準備</h3>
<p>Next.jsを使用するため、Node.js v18以上が必要です。nvmを使ってインストールすることをおすすめします。</p>
<pre><code class="language-bash">&lt;h1&gt;nvmのインストール&lt;/h1&gt;
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.5/install.sh | bash
&lt;h1&gt;Node.js v20をインストール&lt;/h1&gt;
nvm install 20
nvm use 20
</code></pre>
<h2>プロジェクトの作成</h2>
<p>OpenCodeを起動し、ターミナルを開きます。以下のコマンドでNext.jsプロジェクトを作成します。</p>
<pre><code class="language-bash">npx create-next-app@latest my-app --typescript --tailwind --eslint
cd my-app
</code></pre>
<p>プロンプトに従い、App Routerを使用するかどうかなど選択します。今回は「Yes」で進めます。</p>
<h2>AI支援機能の活用</h2>
<h3>コード生成</h3>
<p>OpenCodeの左サイドバーにあるAIアイコンをクリックし、チャットパネルを開きます。例えば「ToDoアプリのリストコンポーネントを作成して」と入力すると、コードを生成してくれます。</p>
<pre><code class="language-typescript">// 生成されたコード例
interface Todo {
  id: number;
  text: string;
  completed: boolean;
}
<p>export default function TodoList() {
  const [todos, setTodos] = useState&lt;Todo[]&gt;([]);
  const [input, setInput] = useState('');</p>
<p>const addTodo = () =&gt; {
    if (input.trim()) {
      setTodos([...todos, { id: Date.now(), text: input, completed: false }]);
      setInput('');
    }
  };</p>
<p>return (
    &lt;div&gt;
      &lt;input value={input} onChange={(e) =&gt; setInput(e.target.value)} /&gt;
      &lt;button onClick={addTodo}&gt;追加&lt;/button&gt;
      &lt;ul&gt;
        {todos.map(todo =&gt; (
          &lt;li key={todo.id}&gt;{todo.text}&lt;/li&gt;
        ))}
      &lt;/ul&gt;
    &lt;/div&gt;
  );
}
</code></pre></p>
<h3>コード補完</h3>
<p>エディタ上でコードを書いていると、自動的に補完候補が表示されます。Tabキーで受け入れられます。特に型定義や繰り返しパターンで効果を発揮します。</p>
<h3>リファクタリング</h3>
<p>コードを選択し、右クリックメニューから「AI: リファクタリング」を選ぶと、改善案を提案してくれます。例えば、上記のToDoリストをよりReactらしくリファクタリングしてもらいましょう。</p>
