f:Td1f,<h2>はじめに</h2>
<p>OpenCodeは、コマンドラインからコード生成・管理を効率化するCLIツールです。本記事では、OpenCode CLIのインストールから基本的なセットアップ手順までを解説します。前提条件として、Node.jsとnpmがインストールされている必要があります。</p>
<h2>前提条件</h2>
<li>Node.js (バージョン14以上推奨)</li>
<li>npm (Node.jsに同梱)</li>
<li>ターミナル（macOS/Linux標準、WindowsではPowerShellなど）</li>
<p>Node.jsが未インストールの場合は、<a href="https://nodejs.org/" target="_blank" rel="noopener">公式サイト</a>からLTS版をダウンロードしてインストールしてください。</p>
<h2>OpenCode CLIのインストール</h2>
<p>OpenCode CLIはnpmパッケージとして公開されています。以下のコマンドでグローバルインストールします。</p>
<pre><code class="language-bash">npm install -g opencode-cli
</code></pre>
<p>インストールが完了したら、次のコマンドでバージョンを確認できます。</p>
<pre><code class="language-bash">opencode --version
</code></pre>
<p>正しくインストールされていれば、バージョン番号が表示されます。</p>
<h3>トラブルシューティング</h3>
<li><strong>パーミッションエラー</strong>: macOS/Linuxで<code>EACCES</code>エラーが出る場合は、<code>sudo</code>を付けて再実行してください。</li>
  <pre><code class="language-bash">  sudo npm install -g opencode-cli
  </code></pre>
<li><strong>コマンドが見つからない</strong>: npmのグローバルインストール先がPATHに通っていない可能性があります。<code>npm root -g</code>でパスを確認し、シェルの設定ファイル（<code>.bashrc</code>や<code>.zshrc</code>）に追加してください。</li>
<h2>初期セットアップ</h2>
<p>インストール後、OpenCodeを使用するための初期設定を行います。</p>
<h3>1. 認証情報の設定</h3>
<p>OpenCodeはAPIを利用するため、認証トークンが必要です。以下のコマンドで設定します。</p>
<pre><code class="language-bash">opencode config set token YOUR_API_TOKEN
</code></pre>
<p>トークンはOpenCodeの管理画面から取得してください。</p>
<h3>2. デフォルトのプロジェクト設定</h3>
<p>作業ディレクトリを指定します。</p>
<pre><code class="language-bash">opencode config set directory ./my-projects
</code></pre>
<h2>基本的な使い方</h2>
<p>セットアップが完了したら、簡単なコマンドを試してみましょう。</p>
<h3>プロジェクトの作成</h3>
<pre><code class="language-bash">opencode create my-first-project
</code></pre>
<h3>テンプレート一覧の表示</h3>
<pre><code class="language-bash">opencode list-templates
</code></pre>
<h3>ヘルプの表示</h3>
<pre><code class="language-bash">opencode --help
</code></pre>
