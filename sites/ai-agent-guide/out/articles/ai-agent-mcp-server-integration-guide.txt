f:T1f45,<h1>AIエージェントとMCPサーバーの連携ガイド：設定・ツール・実践例</h1>
<h2>はじめに</h2>
<p>AIエージェントは、自律的にタスクを実行するプログラムです。その能力を最大限に引き出すには、外部サービスやデータとの連携が欠かせません。そこで注目されているのが<strong>MCP（Model Context Protocol）</strong> です。MCPは、AIエージェントと外部ツール・サーバーを標準化された方法で接続するプロトコルです。本記事では、MCPサーバーの基本から設定、連携手順までを具体的に解説します。</p>
<h2>AIエージェントとは</h2>
<p>AIエージェントは、大規模言語モデル（LLM）を基盤とし、ユーザーの指示に基づいて自律的に行動するソフトウェアです。例えば、Web検索、データ分析、メール送信、ファイル操作などを、ユーザーに代わって実行します。代表的なAIエージェントフレームワークには、LangChain、AutoGPT、BabyAGIなどがあります。</p>
<h2>MCP（Model Context Protocol）とは</h2>
<p>MCPは、Anthropic社が提唱するオープンなプロトコルで、AIエージェントと外部ツール・データソースを安全かつ効率的に連携させるための規格です。従来、各エージェントは独自の方法で外部連携を行っていましたが、MCPにより統一的なインターフェースが提供されます。</p>
<h3>MCPの主な要素</h3>
<li><strong>MCPサーバー</strong>: 外部ツールやデータへのアクセスを提供するサーバー</li>
<li><strong>MCPクライアント</strong>: AIエージェント内で動作し、MCPサーバーと通信するクライアント</li>
<li><strong>ツール</strong>: MCPサーバーが公開する個別の機能（例：計算機、データベース検索、天気API）</li>
<h2>MCPサーバーの設定方法</h2>
<p>MCPサーバーを設定するには、以下の手順を行います。ここでは、Pythonを使用した簡単な例を示します。</p>
<h3>1. 必要なライブラリのインストール</h3>
<pre><code class="language-bash">pip install mcp
</code></pre>
<h3>2. 基本的なMCPサーバーの作成</h3>
<pre><code class="language-python">from mcp.server import Server
from mcp.types import Tool, TextContent
<p>&lt;h1&gt;サーバーインスタンス作成&lt;/h1&gt;
server = Server("example-server")</p>
<p>&lt;h1&gt;ツールの定義&lt;/h1&gt;
@server.list_tools()
async def list_tools() -&gt; list[Tool]:
    return [
        Tool(
            name="calculator",
            description="Perform basic arithmetic operations",
            inputSchema={
                "type": "object",
                "properties": {
                    "expression": {"type": "string"}
                },
                "required": ["expression"]
            }
        )
    ]</p>
<p>&lt;h1&gt;ツールの実行&lt;/h1&gt;
@server.call_tool()
async def call_tool(name: str, arguments: dict) -&gt; list[TextContent]:
    if name == "calculator":
        expression = arguments["expression"]
        result = eval(expression)  # 注意: 本番環境では安全な評価方法を使用
        return [TextContent(type="text", text=str(result))]</p>
<p>&lt;h1&gt;サーバー起動&lt;/h1&gt;
if __name__ == "__main__":
    server.run()
</code></pre></p>
<p>このコードは、計算機ツールを提供するMCPサーバーです。<code>eval</code>は危険なため、本番では<code>ast.literal_eval</code>など安全な方法を使ってください。</p>
<h3>3. サーバーの起動</h3>
<pre><code class="language-bash">python server.py
</code></pre>
<p>デフォルトでは、<code>localhost:8000</code>で待ち受けます。</p>
<h2>AIエージェントとの連携</h2>
<p>AIエージェントがMCPサーバーと連携するには、MCPクライアントを使用します。以下は、LangChainを使った例です。</p>
<h3>LangChainでのMCPクライアント設定</h3>
<pre><code class="language-python">from langchain.agents import initialize_agent, AgentType
from langchain.llms import OpenAI
from langchain_mcp import MCPClient
<p>&lt;h1&gt;MCPクライアントの作成&lt;/h1&gt;
client = MCPClient("http://localhost:8000")</p>
<p>&lt;h1&gt;ツールの取得&lt;/h1&gt;
tools = client.get_tools()</p>
<p>&lt;h1&gt;LLMの設定&lt;/h1&gt;
llm = OpenAI(temperature=0)</p>
<p>&lt;h1&gt;エージェントの初期化&lt;/h1&gt;
agent = initialize_agent(tools, llm, agent=AgentType.ZERO_SHOT_REACT_DESCRIPTION, verbose=True)</p>
<p>&lt;h1&gt;実行&lt;/h1&gt;
agent.run("What is 123 plus 456?")
</code></pre></p>
<p>これにより、AIエージェントがMCPサーバーの計算機ツールを使って計算を行います。</p>
<h2>実践的な連携例</h2>
<h3>天気情報APIとの連携</h3>
<p>MCPサーバーで天気APIをラップし、AIエージェントから利用する例です。</p>
<p><strong>MCPサーバー側（weather_server.py）</strong></p>
<pre><code class="language-python">import httpx
from mcp.server import Server, Tool, TextContent
<p>server = Server("weather-server")</p>
<p>@server.list_tools()
async def list_tools():
    return [
        Tool(
            name="get_weather",
            description="Get current weather for a city",
            inputSchema={
                "type": "object",
                "properties": {
                    "city": {"type": "string"}
                },
                "required": ["city"]
            }
        )
    ]</p>
<p>@server.call_tool()
async def call_tool(name: str, arguments: dict):
    if name == "get_weather":
        city = arguments["city"]
        async with httpx.AsyncClient() as client:
            response = await client.get(f"https://api.openweathermap.org/data/2.5/weather?q={city}&amp;appid=YOUR_API_KEY")
            data = response.json()
            weather = data["weather"][0]["description"]
            temp = data["main"]["temp"] - 273.15
            return [TextContent(type="text", text=f"Weather in {city}: {weather}, temperature: {temp:.1f}°C")]</p>
<p>server.run()
</code></pre></p>
<p><strong>AIエージェント側</strong></p>
<pre><code class="language-python">from langchain.agents import initialize_agent, AgentType
from langchain.llms import OpenAI
from langchain_mcp import MCPClient
<p>client = MCPClient("http://localhost:8000")
tools = client.get_tools()
llm = OpenAI(temperature=0)
agent = initialize_agent(tools, llm, agent=AgentType.ZERO_SHOT_REACT_DESCRIPTION, verbose=True)
agent.run("What's the weather in Tokyo?")
</code></pre></p>
<h2>注意点とベストプラクティス</h2>
<li><strong>セキュリティ</strong>: MCPサーバーはネットワーク上に公開する場合、認証と暗号化を実装してください。特に<code>eval</code>のような危険な関数は避けましょう。</li>
<li><strong>エラーハンドリング</strong>: ツール実行時のエラーを適切に処理し、AIエージェントにわかりやすいメッセージを返すことが重要です。</li>
<li><strong>ツールの粒度</strong>: ツールは単一責任の原則に従い、小さくシンプルに設計すると、AIエージェントが使いやすくなります。</li>
<li><strong>レート制限</strong>: 外部APIを呼び出す場合は、レート制限を考慮し、必要に応じてキャッシュを導入しましょう。</li>
