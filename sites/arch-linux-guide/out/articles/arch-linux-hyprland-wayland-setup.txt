11:T16f3,<h2>目標</h2>
<p>このチュートリアルでは、Arch Linux上にHyprland Waylandコンポジターをインストールし、基本的なデスクトップ環境を構築します。waybar（ステータスバー）、wofi（アプリランチャー）、ドットファイル管理の基礎も学びます。最終的には、軽量でモダンなWaylandデスクトップが使えるようになります。</p>
<h2>前提条件</h2>
<li>Arch Linuxがインストール済みであること（ベースシステムでOK）</li>
<li>インターネット接続があること</li>
<li>基本的なターミナル操作ができること</li>
<li>非rootユーザーが存在し、sudo権限があること</li>
<h2>Step-by-step</h2>
<h3>1. システムの更新と必要なパッケージのインストール</h3>
<p>まずシステムを最新にし、Hyprlandと関連ツールをインストールします。</p>
<pre><code class="language-bash">sudo pacman -Syu
sudo pacman -S hyprland waybar wofi git
</code></pre>
<li><code>hyprland</code>: Waylandコンポジター本体</li>
<li><code>waybar</code>: ステータスバー</li>
<li><code>wofi</code>: アプリランチャー</li>
<li><code>git</code>: ドットファイル管理に使用</li>
<h3>2. Hyprlandの設定ファイルを作成</h3>
<p>設定ファイルは <code>~/.config/hypr/hyprland.conf</code> に配置します。</p>
<pre><code class="language-bash">mkdir -p ~/.config/hypr
nano ~/.config/hypr/hyprland.conf
</code></pre>
<p>最小限の設定例：</p>
<pre><code class="language-conf">&lt;h1&gt;モニター設定（例：1920x1080）&lt;/h1&gt;
monitor=,preferred,auto,1
<p>&lt;h1&gt;キーバインド&lt;/h1&gt;
bind=SUPER,Return,exec,foot
bind=SUPER,D,exec,wofi --show drun
bind=SUPER,Space,exec,wofi --show run
bind=SUPER,Q,killactive,</p>
<p>&lt;h1&gt;起動時に実行するアプリ&lt;/h1&gt;
exec-once=waybar
</code></pre></p>
<p><strong>注意</strong>: <code>exec-once</code> はHyprland起動時に一度だけ実行されます。<code>exec</code> は毎回実行。</p>
<h3>3. waybarの設定</h3>
<p>waybarの設定ファイルを作成します。</p>
<pre><code class="language-bash">mkdir -p ~/.config/waybar
nano ~/.config/waybar/config.jsonc
</code></pre>
<p>最小限の設定例：</p>
<pre><code class="language-jsonc">{
    "layer": "top",
    "position": "top",
    "modules-left": ["hyprland/workspaces"],
    "modules-center": ["clock"],
    "modules-right": ["pulseaudio", "network", "cpu", "memory", "tray"]
}
</code></pre>
<p>スタイル設定も必要です。</p>
<pre><code class="language-bash">nano ~/.config/waybar/style.css
</code></pre>
<pre><code class="language-css">* {
    border: none;
    border-radius: 0;
    font-family: "sans-serif";
    font-size: 13px;
    min-height: 0;
}
<p>window#waybar {
    background: rgba(0, 0, 0, 0.8);
    color: white;
}</p>
<p>#workspaces button {
    padding: 0 5px;
    background: transparent;
    color: white;
}</p>
<p>#workspaces button.focused {
    background: #64727D;
}
</code></pre></p>
<h3>4. wofiの設定（オプション）</h3>
<p>wofiの設定ファイルは <code>~/.config/wofi/config</code> です。</p>
<pre><code class="language-bash">mkdir -p ~/.config/wofi
nano ~/.config/wofi/config
</code></pre>
<pre><code class="language-">width=400
height=300
location=center
show=drun
</code></pre>
<h3>5. ドットファイルの管理（Git）</h3>
<p>ドットファイルをGitで管理する例：</p>
<pre><code class="language-bash">cd ~
git init dotfiles
cd dotfiles
cp -r ~/.config/hypr ~/.config/waybar ~/.config/wofi .
git add .
git commit -m "Initial dotfiles"
</code></pre>
<p>リモートリポジトリにプッシュするには：</p>
<pre><code class="language-bash">git remote add origin &lt;your-repo-url&gt;
git push -u origin main
</code></pre>
<h3>6. Hyprlandの起動</h3>
<p>ログインマネージャーを使わずに起動する場合、ターミナルから直接実行します。</p>
<pre><code class="language-bash">Hyprland
</code></pre>
<p>または、<code>~/.bash_profile</code> に以下を追加して自動ログイン：</p>
<pre><code class="language-bash">if [ -z "${WAYLAND_DISPLAY}" ] &amp;&amp; [ "$(tty)" = "/dev/tty1" ]; then
    exec Hyprland
fi
</code></pre>
<h2>動作確認</h2>
<li>Hyprlandが起動することを確認</li>
<li><code>SUPER+Return</code> でターミナル（foot）が開く</li>
<li><code>SUPER+D</code> でwofiのアプリランチャーが表示される</li>
<li>waybarが画面上部に表示され、ワークスペース、時計、音量などが表示される</li>
<li>ワークスペースを切り替えられる（<code>SUPER+1</code> など、デフォルトで割り当てあり）</li>
