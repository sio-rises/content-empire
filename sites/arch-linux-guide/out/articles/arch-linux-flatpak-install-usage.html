Arch LinuxでFlatpakを導入してサンドボックスアプリを使う方法

Arch LinuxにFlatpakをインストールし、Flathubからサンドボックス化されたアプリケーションを安全に利用する手順を詳しく解説。

Arch Linuxflatpakサンドボックスアプリflathub2026/5/25

はじめに

Flatpakは、Linuxデスクトップ向けのアプリケーションパッケージ管理・サンドボックス化技術です。Arch Linuxでも公式リポジトリから簡単に導入でき、Flathubなどのリモートリポジトリから様々なアプリをインストールできます。本記事では、Arch LinuxにFlatpakをインストールし、実際にアプリを利用するまでの手順を詳しく解説します。

Flatpakのインストール

まず、Flatpak本体をインストールします。

sudo pacman -S flatpak

インストールが完了したら、バージョンを確認します。

flatpak --version

Flathubの追加

Flatpakの主要なリモートリポジトリであるFlathubを追加します。

flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

--if-not-exists は、既に追加済みの場合にエラーを避けるオプションです。

アプリの検索とインストール

アプリの検索

インストールしたいアプリを検索するには、flatpak search コマンドを使います。

flatpak search <アプリ名>

例えば、GIMPを検索する場合：

flatpak search gimp

アプリのインストール

検索結果に表示されるApplication IDを使ってインストールします。

flatpak install flathub <Application ID>

例：GIMPをインストール

flatpak install flathub org.gimp.GIMP

インストール済みアプリの確認

flatpak list

アプリの実行

インストールしたアプリは、通常のコマンドラインからも、デスクトップ環境のランチャーからも起動できます。

コマンドラインから起動する場合：

flatpak run <Application ID>

例：

flatpak run org.gimp.GIMP

アプリの更新

全てのFlatpakアプリを更新するには：

flatpak update

特定のアプリだけ更新する場合：

flatpak update <Application ID>

アプリの削除

不要になったアプリは以下のコマンドで削除します。

flatpak uninstall <Application ID>

また、使用されていないランタイムを削除してディスク容量を節約できます。

flatpak uninstall --unused

トラブルシューティング

パーミッションエラー

Flatpakアプリが特定のファイルやデバイスにアクセスできない場合、flatpak override コマンドでパーミッションを調整します。

例：ホームディレクトリへのアクセスを許可

flatpak override --user --filesystem=home <Application ID>

日本語入力ができない

Flatpakアプリ内で日本語入力が効かない場合、必要なインプットメソッドモジュールをインストールします。

flatpak install flathub org.freedesktop.Platform.Ime.fcitx5

その後、アプリを再起動してください。

ネットワーク接続の問題

一部のアプリがネットワークにアクセスできない場合、--socket=network オプションを追加します。

flatpak override --user --socket=network <Application ID>

まとめ

Arch LinuxにFlatpakを導入することで、サンドボックス化されたアプリケーションを安全に利用できるようになります。Flathubから豊富なアプリを簡単にインストールでき、システムのパッケージ管理と分離されているため、システムの安定性を損なうリスクが低くなります。本記事で紹介した手順を参考に、ぜひFlatpakを活用してみてください。
