2:T13d4,<h2>目標</h2>
<p>このチュートリアルでは、Arch LinuxにKDE Plasmaデスクトップ環境をインストールし、sddmをディスプレイマネージャとして設定、さらに基本的なカスタマイズを行うまでを学びます。最終的には、美しく機能的なデスクトップを手に入れることが目標です。</p>
<h2>前提条件</h2>
<li>Arch Linuxがインストール済みであること（ベースシステムのみでOK）</li>
<li>インターネット接続が可能であること</li>
<li>root権限またはsudoが使える一般ユーザーであること</li>
<li>基本的なコマンド操作（pacman, systemctlなど）が理解できていること</li>
<h2>Step-by-step</h2>
<h3>1. システムのアップデート</h3>
<p>まずはシステム全体を最新の状態にします。</p>
<pre><code class="language-bash">sudo pacman -Syu
</code></pre>
<h3>2. KDE Plasmaのインストール</h3>
<p>KDE Plasmaグループをインストールします。これにより、Plasmaデスクトップ、KDEアプリケーション、必要なライブラリがまとめてインストールされます。</p>
<pre><code class="language-bash">sudo pacman -S plasma
</code></pre>
<p>インストール中にいくつかの選択肢が表示されますが、デフォルトのまま進めて構いません。</p>
<h3>3. sddmのインストールと設定</h3>
<p>Plasmaの推奨ディスプレイマネージャはSDDMです。インストールして有効化します。</p>
<pre><code class="language-bash">sudo pacman -S sddm
sudo systemctl enable sddm
</code></pre>
<p>これで起動時にSDDMが自動起動するようになります。</p>
<h3>4. ネットワーク管理の設定</h3>
<p>デスクトップ環境でネットワークを管理するために、NetworkManagerをインストールし有効化します。</p>
<pre><code class="language-bash">sudo pacman -S networkmanager
sudo systemctl enable NetworkManager
</code></pre>
<h3>5. 再起動してPlasmaを起動</h3>
<p>設定が完了したら、システムを再起動します。</p>
<pre><code class="language-bash">sudo reboot
</code></pre>
<p>再起動後、SDDMのログイン画面が表示されます。ユーザー名とパスワードを入力してログインすると、KDE Plasmaデスクトップが起動します。</p>
<h2>動作確認</h2>
<li><strong>デスクトップが表示されること</strong>: ログイン後、Plasmaデスクトップが正常に表示されることを確認します。</li>
<li><strong>アプリケーションの起動</strong>: アプリケーションメニューから「Dolphin」（ファイルマネージャ）や「Konsole」（ターミナル）が起動できることを確認します。</li>
<li><strong>ネットワーク接続</strong>: システムトレイのネットワークアイコンからWi-Fiや有線接続が設定できることを確認します。</li>
<li><strong>サウンド</strong>: スピーカーアイコンから音量調整ができることを確認します。</li>

<h2>発展: カスタマイズ</h2>
<p>KDE Plasmaは非常にカスタマイズ性が高いです。以下にいくつかの基本的なカスタマイズ方法を紹介します。</p>
<h3>テーマの変更</h3>
<p>システム設定 → 「外観」 → 「Plasmaのスタイル」から、好みのテーマを選択できます。新しいテーマは「新しいPlasmaのスタイルを入手」からオンラインで追加可能です。</p>
<h3>ウィジェットの追加</h3>
<p>デスクトップを右クリック → 「ウィジェットを追加」から、時計、システムモニター、天気予報などのウィジェットをデスクトップに配置できます。</p>
<h3>ショートカットキーの設定</h3>
<p>システム設定 → 「ショートカット」 → 「ショートカットキー」から、アプリケーション起動やウィンドウ操作のショートカットを自由に割り当てられます。</p>
<h3>仮想デスクトップの活用</h3>
<p>システム設定 → 「仮想デスクトップ」で、複数のデスクトップを設定し、ワークスペースを整理できます。</p>
<h3>SDDMのテーマ変更</h3>
<p>SDDMのログイン画面もカスタマイズ可能です。パッケージをインストール:</p>
<pre><code class="language-bash">sudo pacman -S sddm-kcm
</code></pre>
<p>その後、システム設定 → 「起動と終了」 → 「SDDM」からテーマを変更できます。</p>
