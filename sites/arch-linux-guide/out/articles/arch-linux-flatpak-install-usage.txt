11:T137b,<h2>はじめに</h2>
<p>Flatpakは、Linuxデスクトップ向けのアプリケーションパッケージ管理・サンドボックス化技術です。Arch Linuxでも公式リポジトリから簡単に導入でき、Flathubなどのリモートリポジトリから様々なアプリをインストールできます。本記事では、Arch LinuxにFlatpakをインストールし、実際にアプリを利用するまでの手順を詳しく解説します。</p>
<h2>Flatpakのインストール</h2>
<p>まず、Flatpak本体をインストールします。</p>
<pre><code class="language-bash">sudo pacman -S flatpak
</code></pre>
<p>インストールが完了したら、バージョンを確認します。</p>
<pre><code class="language-bash">flatpak --version
</code></pre>
<h2>Flathubの追加</h2>
<p>Flatpakの主要なリモートリポジトリであるFlathubを追加します。</p>
<pre><code class="language-bash">flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
</code></pre>
<p><code>--if-not-exists</code> は、既に追加済みの場合にエラーを避けるオプションです。</p>
<h2>アプリの検索とインストール</h2>
<h3>アプリの検索</h3>
<p>インストールしたいアプリを検索するには、<code>flatpak search</code> コマンドを使います。</p>
<pre><code class="language-bash">flatpak search &lt;アプリ名&gt;
</code></pre>
<p>例えば、GIMPを検索する場合：</p>
<pre><code class="language-bash">flatpak search gimp
</code></pre>
<h3>アプリのインストール</h3>
<p>検索結果に表示されるApplication IDを使ってインストールします。</p>
<pre><code class="language-bash">flatpak install flathub &lt;Application ID&gt;
</code></pre>
<p>例：GIMPをインストール</p>
<pre><code class="language-bash">flatpak install flathub org.gimp.GIMP
</code></pre>
<h3>インストール済みアプリの確認</h3>
<pre><code class="language-bash">flatpak list
</code></pre>
<h2>アプリの実行</h2>
<p>インストールしたアプリは、通常のコマンドラインからも、デスクトップ環境のランチャーからも起動できます。</p>
<p>コマンドラインから起動する場合：</p>
<pre><code class="language-bash">flatpak run &lt;Application ID&gt;
</code></pre>
<p>例：</p>
<pre><code class="language-bash">flatpak run org.gimp.GIMP
</code></pre>
<h2>アプリの更新</h2>
<p>全てのFlatpakアプリを更新するには：</p>
<pre><code class="language-bash">flatpak update
</code></pre>
<p>特定のアプリだけ更新する場合：</p>
<pre><code class="language-bash">flatpak update &lt;Application ID&gt;
</code></pre>
<h2>アプリの削除</h2>
<p>不要になったアプリは以下のコマンドで削除します。</p>
<pre><code class="language-bash">flatpak uninstall &lt;Application ID&gt;
</code></pre>
<p>また、使用されていないランタイムを削除してディスク容量を節約できます。</p>
<pre><code class="language-bash">flatpak uninstall --unused
</code></pre>
<h2>トラブルシューティング</h2>
<h3>パーミッションエラー</h3>
<p>Flatpakアプリが特定のファイルやデバイスにアクセスできない場合、<code>flatpak override</code> コマンドでパーミッションを調整します。</p>
<p>例：ホームディレクトリへのアクセスを許可</p>
<pre><code class="language-bash">flatpak override --user --filesystem=home &lt;Application ID&gt;
</code></pre>
<h3>日本語入力ができない</h3>
<p>Flatpakアプリ内で日本語入力が効かない場合、必要なインプットメソッドモジュールをインストールします。</p>
<pre><code class="language-bash">flatpak install flathub org.freedesktop.Platform.Ime.fcitx5
</code></pre>
<p>その後、アプリを再起動してください。</p>
<h3>ネットワーク接続の問題</h3>
<p>一部のアプリがネットワークにアクセスできない場合、<code>--socket=network</code> オプションを追加します。</p>
<pre><code class="language-bash">flatpak override --user --socket=network &lt;Application ID&gt;
</code></pre>
